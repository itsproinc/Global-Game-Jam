Bootstrap 4

Bootstrap 4 adalah sebuah framework atau alat bantu (yang saat ini masih dalam versi beta) terbaru dari Twitter Bootstrap. Dikatakan Twitter Boostrap, adalah karena pada saat itu, Twitter membuat sebuah framework yang fungsinya untuk mempermudah pembangunan dan pengembangan website mereka. Akhirnya terciptalah boostrap. Tidak lama setelah itu, Twitter akhirnya menyebarkan kerangka kerja ini ke publik dan merubah namanya dari Twitter Bootstrap, menjadi Bootstrap.

Apa fungsinya Bootstrap? Yaitu untuk mempercepat baik pembuatan maupun pengembangan halaman website.

Adapun perubahan dari bootstrap 4 terutama pada perubahan kelas dan perilaku yang lebih spesifik terhadap komponen yang relevan.

Perubahan Global
  • Flexbox diaktifkan secara default. Secara umum ini berarti pindah dari Float dan di seluruh komponen.
  • Beralih dari Less to Sass untuk source CSS.
  • Beralih dari px ke rem sebagai unit CSS utama, meskipun piksel masih digunakan untuk media query dan perilaku grid karena viewports tidak terpengaruh oleh jenis ukuran.
  • Ukuran huruf global meningkat dari 14px menjadi 16px.
  • Menambahkan tier grid baru untuk perangkat yang lebih kecil di 576px dan di bawahnya (tingkat xs/extra small baru).
  • Mengganti tema opsional terpisah dengan opsi yang dapat dikonfigurasi melalui variabel SCSS(misalnya,$enable-gradients: true).

Instalasi Bootstrap

Sebelum kita menggunakan Bootstrap. Berikut adalah langkah-langkah installasinya:

  1. Download file Boostrap dari sini: Boostrap 4
  2. Buat folder dengan nama ITW_SHIFT_NRP (Contoh: ITW_KAMIS10_163040025)
  3. Ekstrak file tersebut kedalam folder folder ITW_SHIFT_NRP (Contoh: ITW_KAMIS10_163040025)
  4. Untuk mendownload JQuery, klik kanan link ini: JQuery lalu pilih "save link as..."
  5. Simpan file JQuery kedalam folder JS di dalam folder asset
